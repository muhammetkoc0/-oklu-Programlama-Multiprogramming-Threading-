Amaç: İşletim sisteminin aynı işlemciyi kullanarak iş parçacıkları arasında hızlı geçiş yapmasını (context-switch) simüle eder.

Çıktı Analizi: Çıktılar (Program 1 ve Program 2) karışık bir şekilde ekrana gelir. Bu, işlemlerin gerçekten aynı anda değil, işletim sistemi tarafından sırayla ama çok hızlı bir şekilde yürütüldüğünü gösterir.
